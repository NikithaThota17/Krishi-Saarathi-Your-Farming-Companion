# Krishi Saarathi - Your Farming Companion

Krishi Saarathi is a Flutter-based Community Service Project developed with the context of Nambur village in mind. The app is designed to support farmers through a simple mobile interface with useful agricultural information in one place.

## Project Overview

The application helps farmers access:
- weather information
- crop advisory
- market details
- government schemes
- AI-based farming assistance
- Telugu and English language support

## Main Features

### Weather Module
- Current weather and forecast
- Location-based weather search
- Farmer-friendly weather view

### Crop Advisory Module
- Crop suggestions based on season, soil, water, and goal
- Best-fit crop recommendation
- Basic farming guidance

### Market Module
- Market-related information and selling support
- State and commodity filtering
- Price insight dashboard

### Schemes Module
- Government schemes for farmers
- Search and category filtering
- Easy-to-read scheme details

### Farmer AI Assistant
- Natural question answering for farming doubts
- Gemini-based assistant support
- Useful for crop, soil, market, and farming questions

## Technology Used

- Flutter
- Dart
- Material Design
- Google Fonts
- Shared Preferences
- HTTP API integration
- Gemini API

## Project Context

This project was developed as part of the Community Service Project for Nambur village, focusing on helping farmers use digital tools more easily and access agricultural support in a practical way.

## How to Run

1. Clone the repository
2. Open the project in Flutter
3. Run:

```powershell
flutter pub get
flutter run
```

## APK Build

To generate an APK:

```powershell
flutter build apk --release
```
